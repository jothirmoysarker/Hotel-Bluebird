***Hotel Management System***

A complete Hotel Management System built using HTML, CSS, JavaScript, PHP, and a fully structured MySQL database.
This project handles employee management, admin control, booking operations, and overall hotel workflow automation.

**Features**

- Admin & User Login System

- Employee Management (Add, edit, delete, view staff)

- Room & Booking Management

- Fully Integrated SQL Database

- Built using HTML, CSS, JavaScript, PHP

**MySQL Database included**

*Requirements:*

- To run this project locally, you need:

- XAMPP (recommended)

- PHP 7+

- MySQL Database

- Web Browser (Chrome, Firefox, etc.)

**How to Install & Run the Project**

Follow the steps below to set up the system:

1. Install XAMPP

Download from
https://www.apachefriends.org/

2. Move the Project to htdocs

Copy the project folder into:

C:\xampp\htdocs\

3. Start Apache & MySQL

Open XAMPP Control Panel
✔ Start Apache
✔ Start MySQL

4. Import the Database

Go to http://localhost/phpmyadmin/

Click Import

Choose the SQL file included in the project

Click Go

Database will be created automatically.

5. Run the Project

Go to your browser and type:

http://localhost/your_project_folder/
